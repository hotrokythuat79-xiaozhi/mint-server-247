All notable changes to this project will be documented in this file.

## [v1.0] - Initial Stable Release

### Added
- Kernel tuning for 24/7 server stability
- HDD optimization (noatime + read-ahead 2MB)
- Persistent udev rule for read-ahead
- Memory guard settings
- Log size limitation (100MB)
- Disable sleep/hibernate targets
- SSH-based Git deployment

### Tested On
- ASUS X451CA
- i3-3217U
- 6GB RAM
- HDD 320GB
- Linux Mint 22.3 Xfce
